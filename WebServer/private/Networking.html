Network Media

Network media is how an electrical signal travels as it moves from one component to another. The three main travel categories are: Copper cable, Wireless and Fibre optics. Copper wires are very old school and is the main reason that a lot of it is being changed to optical fibres which are cheaper and much quicker. The bandwidth for an optical fibre is 75THz whereas copper wires only have bandwidth of 100MHz which is a lot less. Wireless signals have the capacity to have a bandwidth of 2.4GHz. However, there a lot of factors which need to be right for a wireless signal to output this much bandwidth. One of the biggest problems with wirelessly transmitting data is that it needs to be secured in order to ensure privacy.


Addressing and Protocols


An IP address is a unique numerical value that is used to identify a computer on a network. There are two kinds of IP addresses, public and private. The public Ip is know as the global unique Ip address, public IPs are assigned by the Internet Assigned Number Authority. In order to understand networking IP addresses and subnetting , you also have to understand binary. The address is made up of 32 binary bits, which can be divisible into a network portion and host portion with the help of a subnet mask.
